## Description

ShepherdStaff is a platform designed to help relieve mental and administrative burdens of youth mentors and shepherds serving in youth ministries. Our objective is that, like the name of our app, our software will be a useful tool that will help you to focus on the actual walking and journeying with youth itself.

The ShepherdStaff backend is written in NestJS, with TypeORM and Postgres.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
$ npm run start:dev
```

## API Documentation

You may access the Swagger documentation here:

```bash
http://localhost:3000/api-docs
```

Note that when you are using the Swagger docs, you will first need to login to get a JWT token to continue using the rest of the endpoints. Scroll down to the `Auth` endpoints, and open the `POST /api/auth/login` endpoint to login. Then copy and paste the returned access token in the `Authorize` button at the top right of the Swagger docs.

If you do not have an account yet, use the `POST /api/mentor` endpoint to sign up (as a mentor)
